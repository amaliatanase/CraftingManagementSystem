\doxysubsection{Detailed Description}
The main Graphical User Interface of the application. 


\begin{DoxyItemize}
\item Inherits from QWidget. It provides a desktop interface using Qt Widgets for users to interact with the Handmade Craft Store inventory. It delegates all business logic and data manipulation to the \doxylink{class_controller}{Controller} layer. 
\end{DoxyItemize}
