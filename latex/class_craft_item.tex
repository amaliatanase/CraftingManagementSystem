\doxysubsection{Detailed Description}
Represents a single product in the handmade craft store. 


\begin{DoxyItemize}
\item This class acts as the main domain entity of the application. It holds all relevant details about a specific craft item, including its identifier, name, category, pricing, and available stock. 
\end{DoxyItemize}
