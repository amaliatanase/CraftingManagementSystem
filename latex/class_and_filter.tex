\doxysubsection{Detailed Description}
A composite filter that applies a logical AND operation between two filters. 


\begin{DoxyItemize}
\item This class uses the Composite Design Pattern to treat a group of filters exactly as a single filter. It evaluates to true only if both child filters evaluate to true. 
\end{DoxyItemize}
