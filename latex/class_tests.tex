\doxysubsection{Detailed Description}
A utility class containing automated tests for the application. 


\begin{DoxyItemize}
\item This class groups together all the unit tests required to validate the core functionality of the \doxylink{class_repository}{Repository} and \doxylink{class_controller}{Controller} layers. It uses standard C++ assertions to ensure that CRUD operations, undo/\+redo mechanisms, and filters work correctly before the graphical interface is launched. 
\end{DoxyItemize}
