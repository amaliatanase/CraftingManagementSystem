\doxysubsection{Detailed Description}
Factory class for creating repository instances. 


\begin{DoxyItemize}
\item This class implements the Factory Design Pattern. It centralizes the creation logic for repository objects, allowing the application to easily switch between different storage formats (like CSV or JSON) without changing the core business logic. 
\end{DoxyItemize}
