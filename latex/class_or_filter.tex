\doxysubsection{Detailed Description}
A composite filter that applies a logical OR operation between two filters. 


\begin{DoxyItemize}
\item This class uses the Composite Design Pattern. It evaluates to true if at least one of the child filters evaluates to true. 
\end{DoxyItemize}
