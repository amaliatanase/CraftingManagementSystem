\doxysubsection{Detailed Description}
Base class for managing the storage and retrieval of \doxylink{class_craft_item}{Craft\+Item} objects. 


\begin{DoxyItemize}
\item This class acts as the lowest layer in the architecture, providing an in-\/memory data structure to manage items. It uses virtual methods to allow derived classes (like CSV or JSON repositories) to override behaviors and implement persistent storage. 
\end{DoxyItemize}
