\doxysubsection{Detailed Description}
Handles the business logic of the application. 


\begin{DoxyItemize}
\item The \doxylink{class_controller}{Controller} acts as a bridge between the User Interface and the \doxylink{class_repository}{Repository}. It manages CRUD operations, handles data validation, executes undo/\+redo functionality using the Command Design Pattern, and applies filtering logic. 
\end{DoxyItemize}
