\doxysubsection{Detailed Description}
Abstract base class (Interface) representing a generic executable command. 


\begin{DoxyItemize}
\item This class is the core of the Command Design Pattern implementation. It encapsulates operations as objects, allowing the \doxylink{class_controller}{Controller} to maintain a stack of generic pointers for polymorphic execution, undo, and redo functionalities. 
\end{DoxyItemize}
