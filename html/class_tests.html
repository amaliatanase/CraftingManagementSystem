Detailed Description

A utility class containing automated tests for the application.

  • This class groups together all the unit tests required to validate the core functionality of the Repository and Controller layers. It uses standard C++ assertions to ensure that CRUD operations, undo/redo mechanisms, and filters work correctly before the graphical interface is launched.
