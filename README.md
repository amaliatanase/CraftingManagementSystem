# Handmade Craft Store Management

A C++ desktop application built with the Qt framework that manages the inventory of a handmade craft store.

## Features
* Three-layered architecture (UI, Controller, Repository)
* CRUD operations
* Undo/Redo functionality (Command Pattern)
* Advanced filtering with AND/OR logical operators (Specification/Composite Pattern)
